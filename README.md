## Установка и запуск (без VR-очков)

1. Установите **Unity 6.4(6000.4.1f1).
2. Склонируйте репозиторий:  
   `git clone https://github.com/ваш-аккаунт/VR-Sandbox-HandTracking.git`
3. Откройте проект в Unity.
4. Убедитесь, что установлены пакеты (через Package Manager):
   - XR Interaction Toolkit (3.x)
   - OpenXR Plugin
   - XR Device Simulator
5. Откройте сцену `Assets/Scenes/MainScene.unity`.
6. Нажмите **Play**.
7. Управление жестами в эмуляции описано ниже.

## Управление в режиме эмуляции (клавиатура + мышь) !Пример!

| Жест (реальный)         | Эмуляция (клавиша) | Действие                          |
|-------------------------|--------------------|-----------------------------------|
| Щипок (thumb + index)   | `G`               | Захватить / отпустить объект      |
| Указательный жест       | `F2`               | Активировать луч телепортации     |
| Щипок во время указания | `F2` + щипок (`F1`)| Телепортироваться в точку луча    |
| Взгляд на левую ладонь + щипок | `G` + `F1`   | Открыть / закрыть системное меню  |

> **Примечание:** Эмуляция рук управляется мышью (зажимайте правую кнопку и двигайте для вращения камеры). Пальцы шевелятся автоматически при нажатии клавиш жестов.

## Структура проекта (основные папки)

- `Assets/Scenes/` – главная сцена `MainScene`
- `Assets/Scripts/Input/` – абстракция ввода (`IHandInput`, `MouseHandInput`)
- `Assets/Scripts/Interaction/` – захват, бросание
- `Assets/Scripts/Locomotion/` – телепортация, движение
- `Assets/Scripts/UI/` – системное меню, указание
- `Docs/` – таблица жестов, баг-трекер, отчёты
